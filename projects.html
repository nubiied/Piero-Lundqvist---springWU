RISE Rube Goldberg Machine

This project consisted of making a step in a Rube Goldberg machine using Arduino. This machine was to be presented to the project leader, which would then connect it alongside 20 other machines. Our machine was a game where you had to match a randomized angle using a potentiometer with the help of a sound which increased its frequency the closer you were to the angle.
